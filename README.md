## Datasets description
Input: Review từ các trang web và forum nói về du lịch Bình Định.

Output: Đánh giá review trên các khía cạnh:
- Dịch vụ giải trí
- Dịch vụ lưu trú 
- Hệ thống nhà hàng 
- Dịch vụ ăn uống
- Dịch vụ di chuyển
- Dịch vụ mua sắm

## Technology
- PhoBERT

- Crawl, label data
